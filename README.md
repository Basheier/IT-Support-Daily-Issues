# IT Support Daily Issues

A structured documentation project for daily IT support problems and solutions.  
This repository helps organize troubleshooting steps, improve response time, and build a knowledge base for recurring issues in Windows, Network, Servers, and Office 365 environments.

---

## 📌 Purpose
- Document daily IT issues and solutions.
- Build a clean and organized knowledge base.
- Improve troubleshooting speed.
- Share knowledge with team members.
- Track recurring problems and their fixes.

---

## 📂 Categories

### 🔹 Windows
Common issues related to printers, network resets, slow performance, and system errors.

### 🔹 Network
DNS failures, IP conflicts, Wi-Fi drops, and general connectivity problems.

### 🔹 Servers
Windows Server errors, VSS writer failures, Active Directory issues, and backup problems.

### 🔹 Office 365
SMTP errors, Outlook sync issues, license problems, and user access cases.

---

## 🛠 How to Use
- Add a new file for each issue you face.
- Write a short description + steps to solve it.
- Update files whenever you find a better solution.
- Use this repository as your daily IT support reference.
