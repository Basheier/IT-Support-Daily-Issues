# Slow Performance Issue

## Problem
The computer is running slowly or freezing during normal tasks.

## Possible Causes
- High CPU or RAM usage.
- Too many startup programs.
- Low disk space.
- Corrupted system files.
- Malware or unwanted background apps.

## Solution
1. Check Task Manager:
   - Ctrl + Shift + Esc
   - Identify apps using high CPU/RAM.

2. Disable startup programs:
   - Task Manager → Startup → Disable unnecessary apps.

3. Clean disk space:
   - Run "Disk Cleanup"
   - Delete temporary files.

4. Run system repair:
   - sfc /scannow
   - DISM /Online /Cleanup-Image /RestoreHealth

5. Scan for malware:
   - Windows Security → Full Scan

6. Restart the device after cleanup.
