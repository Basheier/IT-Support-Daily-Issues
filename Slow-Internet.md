# Slow Internet Issue

## Problem
Internet speed is very slow or pages take too long to load.

## Possible Causes
- Weak WiFi signal.
- Network congestion (too many users).
- ISP speed limitations.
- Faulty router or switch.
- Background apps consuming bandwidth.
- DNS issues.

## Solution
1. Test internet speed:
   - Use speedtest.net

2. Restart router and modem.

3. Check signal strength:
   - Move closer to router
   - Avoid walls and interference

4. Change DNS:
   - Google DNS: 8.8.8.8 / 8.8.4.4
   - Cloudflare DNS: 1.1.1.1

5. Close background apps:
   - Task Manager → Processes → End high bandwidth apps

6. Check network congestion:
   - Disconnect unused devices

7. Contact ISP if speed is below plan.
