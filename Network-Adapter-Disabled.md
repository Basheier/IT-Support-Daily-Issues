# Network Adapter Disabled Issue

## Problem
Network adapter is disabled, causing loss of internet or LAN connectivity.

## Possible Causes
- Adapter manually disabled.
- Corrupted driver.
- Windows update conflict.
- Power-saving settings turning off the adapter.

## Solution
1. Enable adapter:
   - Control Panel → Network & Sharing Center
   - Change adapter settings
   - Right-click adapter → Enable

2. Update driver:
   - Device Manager → Network adapters → Update driver

3. Disable power saving:
   - Device Manager → Network adapter → Properties
   - Power Management → Uncheck "Allow the computer to turn off this device"

4. Reset network settings:
   - Settings → Network & Internet → Advanced network settings → Network reset

5. Restart device.
