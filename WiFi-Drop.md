# WiFi Drop Issue

## Problem
WiFi disconnects frequently or shows limited connectivity.

## Possible Causes
- Weak signal or interference.
- Router overheating.
- Incorrect WiFi adapter settings.
- Outdated network drivers.
- Too many devices connected to the router.

## Solution
1. Restart router and modem.
2. Forget WiFi network and reconnect.
3. Update WiFi adapter driver:
   - Device Manager → Network adapters → Update driver
4. Change WiFi frequency:
   - Try switching between 2.4 GHz and 5 GHz
5. Reduce interference:
   - Move closer to router
   - Avoid walls or microwave interference
6. Test connection:
   - ping 8.8.8.8 -t
