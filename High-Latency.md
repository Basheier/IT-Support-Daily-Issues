# High Latency Issue

## Problem
Internet connection has high latency (high ping), causing delays in video calls, gaming, and online meetings.

## Possible Causes
- Weak WiFi signal.
- Network congestion.
- ISP routing issues.
- Background downloads or updates.
- Faulty router or switch.
- Long-distance servers.

## Solution
1. Test latency:
   - ping 8.8.8.8 -t

2. Switch from WiFi to Ethernet (if possible).

3. Close background apps:
   - Task Manager → End high bandwidth apps

4. Restart router or modem.

5. Change DNS:
   - Google DNS: 8.8.8.8 / 8.8.4.4
   - Cloudflare DNS: 1.1.1.1

6. Reduce network congestion:
   - Disconnect unused devices

7. Contact ISP if latency is consistently high.
