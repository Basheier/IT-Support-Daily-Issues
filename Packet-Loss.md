# Packet Loss Issue

## Problem
Internet connection is unstable, video calls freeze, or downloads fail due to packet loss.

## Possible Causes
- Weak WiFi signal.
- Overloaded router or switch.
- Faulty Ethernet cable.
- ISP network congestion.
- High latency routes.
- Background apps consuming bandwidth.

## Solution
1. Test packet loss:
   - ping 8.8.8.8 -t
   - Look for "Request timed out"

2. Restart router or modem.

3. Switch from WiFi to Ethernet (if possible).

4. Replace Ethernet cable or try another switch port.

5. Close background apps:
   - Task Manager → End high bandwidth apps

6. Change DNS:
   - Google DNS: 8.8.8.8 / 8.8.4.4

7. Contact ISP if packet loss is consistent.
